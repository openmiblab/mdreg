mdreg 0.5.0
===========

Major update with some changes that are NOT backwards compatible. 


Main features added
-------------------

- Add zarr support
- Add dask for parallel computing
- Optional installation of plot and coreg packages
- Change defaults for coregistration: default=skimage
- Support for ants registration package
- Data on zenodo rather than included. 
- Extended documentation with user guide
- Activity tracking on website to understand uptake
- More examples and tutorials added
