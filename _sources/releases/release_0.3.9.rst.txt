mdreg 0.3.9
===========
We're happy to announce the release of mdreg 0.3.9!

Summary
--------

This is the first stable release, deployed in clinical studies iBEAt, AFiRM and TRISTAN.

This version uses an object oriented interface for the main functionality, which will be deprecated in version 0.4 and replaced by a simpler functional interface.


Enhancements

- Mask, parallel and log flags as attributes of the MDReg class (`#20 <https://github.com/QIB-Sheffield/mdreg/pull/20>`_).
